🌦️ Weather App using JavaScript, Tailwind CSS & OpenWeatherMap API

A simple and responsive Weather App built using HTML, Tailwind CSS, and JavaScript.

This project uses the OpenWeatherMap API to fetch real-time weather information based on city name input.

## 🚀 Features
- Search weather by city name
- Real-time temperature display
- Weather description
- Clean and responsive UI using Tailwind CSS
- Error handling for invalid city names

## 🛠️ Tech Stack
- HTML
- Tailwind CSS
- JavaScript (ES6)
- OpenWeatherMap API

## 📡 API Used
- OpenWeatherMap: https://openweathermap.org/api

## 📌 Purpose
This project was built to improve my JavaScript skills, API handling, and frontend design using Tailwind CSS.
